Hashing function for strings that generates the sting hash as System.UInt32

public static uint StableHash(string s)
Method parameters
string s
String to hash

Return value

uint
Hash of given string

Remarks

The hash function is based on the .NET implementation of System.String.GetHashCode optimized for 64bit processors. The reason for the explicit hash function is that the .NET implementation is not granted to be stable across versions and/or frameworks. The stability is needed as the Base36 encoded hash is a part of the file name and may be used in perma links
